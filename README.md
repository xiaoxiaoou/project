### 项目开启
1、npm install<br/> 
2、npm run dev<br/>
3、浏览器中打开（监听的是8080端口）<br/>
### 项目开发
使用vue搭配其他开发工具，引用jQuery-ui等实现拖拽、排序等功能；<br/>
该项目还包含了基本的增删改查功能。
